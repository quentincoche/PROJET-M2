create a new repository on the command line :

git add README.md
git commit -m "first commit"
git branch -M master
git remote add origin https://github.com/quentincoche/PROJET-M2.git
git push -u origin master

git remote add origin https://github.com/quentincoche/PROJET-M2.git
git branch -M master
git push -u origin master
